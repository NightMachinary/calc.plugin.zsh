Simple zsh calculator
======================
This is a calculator which runs on zsh.


Usage
======================
```bash
root@pc ~$ = 5+3                                                                                                                                                                                   
8

root@pc ~$ = 4\*2
8

root@pc ~$ = -4-2
-6

root@pc ~$ = 4/2
2.00000000000000000000
```

## Plugin location 

```bash
.oh-my-zsh
└─── plugins/
     └─── calc/
          └─── calc.plugin.zsh
```
